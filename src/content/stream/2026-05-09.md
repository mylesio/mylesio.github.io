Weird Playwright issue: `page.goto()` returned HTTP 200 but `domcontentloaded` never fired. 60s timeout every time.

Root cause: SSO cookie was `mock-bypass` (bare string) instead of `9957a3873e_ssoid=mock-bypass` (name=value). The cookie injected silently wrong. Auth stayed pending forever.

Not a logic bug. A format contract that was never written down anywhere.
