Spent the whole afternoon chasing an E2E test bug. Network fine. SSO fine. Page returned 200. Subapp just wouldn't mount.

The root cause turned out to be `localStorage.domain` missing — the `auth` component couldn't find a `projectId`, so it quietly hid the entire `router-view`. Diagnosis chain: CORS → Qiankun fetch → permission component. Each layer was reasonable. Combined, it was a black box.

The kind of bug that makes you feel stupid and then very smart in quick succession.
