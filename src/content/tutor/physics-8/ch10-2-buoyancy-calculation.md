## 🏐 热身

浮力综合题是中考最难、最综合的计算题之一！它结合了密度、重力、压力、浮沉……就像排球比赛的决胜局，需要把所有技能都用上。今天我们来逐类型拆解！

---

## 📚 核心内容

### 一、浮力四种计算方法对比

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| 阿基米德原理法 | $F_{浮} = \rho_{液}gV_{排}$ | 已知液体密度和排液体积 |
| 称重法 | $F_{浮} = G - F_{示}$ | 已知空气中重力和液中弹簧秤示数 |
| 漂浮平衡法 | $F_{浮} = G_{物}$ | 物体漂浮或悬浮 |
| 压强差法 | $F_{浮} = p_{下}S - p_{上}S$ | 理解原理用，计算少用 |

### 二、典型综合题精讲

**题型1：求物体密度（浮力法）**

已知：木块漂浮在水中，用弹簧秤在空气中称重G=3N，在水中称重F示=0（漂浮，弹簧秤读数为0），水的密度1000kg/m³，g=10N/kg。

漂浮：$F_{浮} = G = 3$ N

$$V_{排} = \frac{F_{浮}}{\rho_{水}g} = \frac{3}{1000 \times 10} = 3 \times 10^{-4} \text{ m}^3$$

若有1/4体积露出水面：$V_{物} = V_{排} / \frac{3}{4} = 4 \times 10^{-4}$ m³

$$\rho_{木} = \frac{m}{V_{物}} = \frac{G/g}{V_{物}} = \frac{3/10}{4 \times 10^{-4}} = 750 \text{ kg/m}^3$$

**题型2：潜水艇沉浮控制**

潜水艇通过改变自身重力（充排水仓的水）来控制浮沉：
- 充水：自重增大 → 重力>浮力 → 下沉
- 排水：自重减小 → 重力<浮力 → 上浮
- 平衡：重力=浮力 → 悬浮

**题型3：盐水浮鸡蛋**

鸡蛋在淡水中沉底（ρ蛋>ρ淡水），加盐后液体密度增大，当ρ液=ρ蛋时悬浮，继续加盐ρ液>ρ蛋时上浮。

**题型4：浮力与密度关系**

$$\frac{V_{露}}{V_{总}} = 1 - \frac{\rho_{物}}{\rho_{液}}$$

例：冰块漂浮在水中，露出水面的体积占总体积的比例？

$$\frac{V_{露}}{V_{总}} = 1 - \frac{\rho_{冰}}{\rho_{水}} = 1 - \frac{0.9}{1.0} = 0.1 = 10\%$$

冰山露出水面约10%，藏在水下90%——"冰山一角"！

---

## 📐 公式总结

| 场景 | 公式 |
|------|------|
| 漂浮 | $F_{浮} = G$，$\\rho_{物} < \\rho_{液}$ |
| 悬浮 | $F_{浮} = G$，$\\rho_{物} = \\rho_{液}$ |
| 下沉（沉底） | $F_{浮} < G$，$\\rho_{物} > \\rho_{液}$ |
| 露出比例 | $V_{露}/V_{总} = 1 - \\rho_{物}/\\rho_{液}$ |

---

## 💡 易错点

1. **漂浮和悬浮的区别**：漂浮时部分在液面以上，悬浮时完全在液体内部
2. **沉底的物体受三力**：重力G（向下）、浮力F浮（向上）、支持力N（向上）
3. **改变形状不改变浮力**：同一物体完全浸没时，无论形状如何，浮力不变（因为V排不变）
4. **同一物体不同液体**：浮力不同（ρ液不同），下沉程度不同
