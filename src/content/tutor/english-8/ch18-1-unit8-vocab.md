### 🔑 完成时标志副词对比

| 副词 | 用法 | 位置 | 例句 |
|------|------|------|------|
| already | 肯定句 | have后/句尾均可 | I've **already** read it. / I've read it **already**. |
| yet | 否定/疑问 | 句尾 | Have you read it **yet**? / I haven't read it **yet**. |
| ever | 疑问句 | 过去分词前 | Have you **ever** been to Beijing? |
| never | 否定含义肯定句形式 | 过去分词前 | I've **never** read that book. |
| just | 肯定句（刚刚） | have后 | She has **just** arrived. |

## 💡 易错点

### 1. yet 和 already 的区别

```
already（已经）= 肯定句，表示"已经完成了"
yet（还/已经）= 疑问句或否定句

✅ I've already eaten. （我已经吃了）
✅ Have you eaten yet? （你吃了吗？）
✅ I haven't eaten yet. （我还没吃）
❌ Have you already eaten?（语气变强，暗示催促，较少用）
```

### 2. read 的三种形式都一样

read / read / read（拼写相同，但读音不同！）

- 原形 read：/riːd/（现在时）
- 过去式 read：/rɛd/（红色 red 的发音！）
- 过去分词 read：/rɛd/

```
I read every day.（现在时，读 /riːd/）
I read yesterday.（过去式，读 /rɛd/）
I have read the book.（完成时，读 /rɛd/）
```

### 3. experience 的两种词性

```
an experience = 一次经历（可数）
experience = 经验（不可数）

She has a lot of experience.（经验丰富，不可数）
It was an amazing experience.（一次经历，可数）
```
