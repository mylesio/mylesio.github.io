## 🏐 热身

排球场地球网中间有个中心点——球网两侧的场地关于这个点完全对称！

还有扑克牌"A"：正着看和倒过来看一模一样。这种"绕中心点转180°后和自身重合"的性质，就是今天要学的**中心对称**！

## 🏐 发球

### 中心对称的定义

把一个图形绕某点旋转180°后，与另一个图形完全重合，则这两个图形关于这个点成**中心对称**，这个点叫做**对称中心**。

### 中心对称的性质

**性质一：** 对称点连线过对称中心

**性质二：** 对称中心是对称点连线的**中点**（最重要！）

$$\text{若}\ A,\ A'\ \text{关于点}\ O\ \text{成中心对称，则}\ O\ \text{是}\ AA'\ \text{的中点}$$

**性质三：** 对应线段平行且相等

### 中心对称图形

一个图形绕某点旋转180°后能与**自身**完全重合，称为**中心对称图形**，这个点叫做对称中心。

| 图形 | 中心对称 | 轴对称 |
|------|---------|--------|
| 圆 | ✅（圆心） | ✅（任意直径） |
| 平行四边形 | ✅（对角线交点） | ❌ |
| 正方形 | ✅ | ✅ |
| 等腰梯形 | ❌ | ✅ |
| 等腰三角形 | ❌ | ✅ |
| 线段 | ✅（中点） | ✅ |

### 例题1：原点对称（最常考）

点 $A(2,3)$ 关于原点 $O(0,0)$ 成中心对称的点 $A'$ 坐标？

> 口诀：**关于原点对称，坐标各取反**

$$A(2,3) \to A'(-2,-3)$$

### 例题2：任意中心求对称点

已知 $A(1,2)$，对称中心为 $O(2,3)$，求 $A'$ 坐标。

因为 $O$ 是 $AA'$ 的**中点**，用中点公式：

$$\frac{1+x_{A'}}{2}=2 \implies x_{A'}=3 \qquad \frac{2+y_{A'}}{2}=3 \implies y_{A'}=4$$

$$\therefore A'=(3,4)$$

> 💡 口诀：**中心坐标×2 − 原点坐标 = 对称点坐标**

### 例题3：判断中心对称图形

判断以下图形是否为中心对称图形：

| 图形 | 结论 |
|------|------|
| 平行四边形 | ✅ 是（对角线交点为中心） |
| 等腰梯形 | ❌ 不是 |
| 正六边形 | ✅ 是（中心点） |
| 字母"S" | ✅ 是 |
| 字母"Z" | ✅ 是 |

## 🏐 扣球

### 中心对称 vs 轴对称

| | 中心对称 | 轴对称 |
|--|---------|--------|
| 操作 | 绕**点**旋转180° | 关于**线**翻折 |
| 对称元素 | 对称中心（点） | 对称轴（线） |
| 对应点关系 | 连线被中心**平分** | 连线被对称轴**垂直平分** |

> ⚠️ 有些图形两种对称都有（圆、正方形），有些只有一种（平行四边形只有中心对称；等腰三角形只有轴对称）。

### ❌ 易错区

| 错误 | 正确 |
|------|------|
| 关于原点对称：$A(3,-2)\to A'(3,2)$（只变y） | $A'(-3,2)$，两个坐标都取反 |
| 认为对称中心是对称点连线的中点需要证明 | 这是**定义**中的性质，直接用！ |
| 等腰梯形是中心对称图形 | 不是，旋转180°后不能与自身重合 |

> 🎯 **中点公式用法：** 已知 $A$、$O$，求 $A'$：$A'=(2x_O - x_A,\ 2y_O - y_A)$
