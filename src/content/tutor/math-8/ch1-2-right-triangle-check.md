## 🏐 热身

排球队要在操场上画一个标准的直角训练区域。教练拿了根绳子，量出 3 米、4 米、5 米三段，说："把绳子拉成三角形，那个最大的角一定是直角。"

等等 🤔 上节课我们知道：**直角三角形 → 满足 $a^2+b^2=c^2$**

但反过来呢？**满足 $a^2+b^2=c^2$ → 一定是直角三角形吗？**

今天就来搞清楚这个"反过来"的问题。

## 🏐 发球

### 勾股定理的逆定理

> **如果三角形的三边 $a$、$b$、$c$ 满足 $a^2 + b^2 = c^2$，那么这个三角形是直角三角形，且 $c$ 所对的角是直角。**

注意关键点：
- **$c$ 必须是最长边**（先把三条边排个大小）
- 验证的是 $\text{两短边}^2 + \text{之和} = \text{最长边}^2$

### 判断步骤（三步走）

1️⃣ 找出最长边，设为 $c$，另外两边为 $a$、$b$

2️⃣ 计算 $a^2 + b^2$ 和 $c^2$

3️⃣ 比较结果：
- $a^2 + b^2 = c^2$ → ✅ 直角三角形
- $a^2 + b^2 > c^2$ → 锐角三角形（最大角是锐角）
- $a^2 + b^2 < c^2$ → 钝角三角形（最大角是钝角）

> 💡 **多出来的两个判断是附赠技能——考试经常考！**

### 例题演练

**例 1（基础）：判断三边为 5, 12, 13 的三角形是什么三角形。**

最长边 $c = 13$

$$5^2 + 12^2 = 25 + 144 = 169 = 13^2$$

等式成立 → **直角三角形** ✅

**例 2（进阶）：三边为 7, 8, 12 的三角形呢？**

最长边 $c = 12$

$$7^2 + 8^2 = 49 + 64 = 113$$
$$12^2 = 144$$

$113 < 144$ → $a^2+b^2 < c^2$ → **钝角三角形** 📐

**例 3（带根号）：三边为 $\sqrt{2}$, $\sqrt{3}$, $\sqrt{5}$ 的三角形呢？**

最长边 $c = \sqrt{5}$

$$(\sqrt{2})^2 + (\sqrt{3})^2 = 2 + 3 = 5 = (\sqrt{5})^2$$

等式成立 → **直角三角形** ✅

> 🏐 排球小知识：标准排球场底线 9 米，中线到攻击线 3 米。想验证某个角是不是直角？量三条边，算一下就知道了！

### 常见易错点

❌ **错误：** 随便拿两条边的平方和跟第三条比

✅ **正确：** 一定要拿 **两条短边** 的平方和跟 **最长边** 的平方比

❌ **错误：** 三边是 6, 8, 11，$6^2 + 8^2 = 100 = 10^2$，所以是直角三角形

✅ **正确：** 最长边是 11 不是 10！$6^2 + 8^2 = 100$，$11^2 = 121$，$100 < 121$，是钝角三角形

### 一张表记住

| 条件 | 结论 | 最大角 |
|:---:|:---:|:---:|
| $a^2+b^2 = c^2$ | 直角△ | 90° |
| $a^2+b^2 > c^2$ | 锐角△ | < 90° |
| $a^2+b^2 < c^2$ | 钝角△ | > 90° |

记忆口诀：**"等于直角，大于尖角，小于钝角"**（两短边平方和跟最长边平方比）
