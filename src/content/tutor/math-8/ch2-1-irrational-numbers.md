## 🏐 热身

排球场地是正方形！标准场地边长9米，从一个角到对角的距离是多少？

$$d = \sqrt{9^2 + 9^2} = \sqrt{162} = 9\sqrt{2}$$

这个 $\sqrt{2}$ 到底是什么数？它能写成分数吗？今天就来搞清楚这个让古希腊数学家抓狂的神秘数字！

## 🏐 发球

### √2 的发现

边长为1的正方形，对角线 $d$ 满足 $d^2 = 1^2 + 1^2 = 2$，所以 $d = \sqrt{2}$。

**$\sqrt{2}$ 能写成分数吗？** 用反证法可以证明：不能。**$\sqrt{2}$ 是一种全新的数！**

### 无理数的定义

**无理数**：无限不循环小数，不能写成 $\dfrac{p}{q}$（$p$、$q$ 为整数，$q \neq 0$）的形式。

| 类型 | 例子 |
|------|------|
| 开方开不尽 | $\sqrt{2},\ \sqrt{3},\ \sqrt{5},\ \sqrt[3]{2}$ |
| 特殊常数 | $\pi \approx 3.14159\ldots$ |
| 构造的无限不循环小数 | $0.1010010001\ldots$ |

**不是无理数的陷阱：**
- $\sqrt{4} = 2$（整数，有理数）
- $\sqrt{0.25} = 0.5$（有理数）
- $\sqrt[3]{-8} = -2$（整数，有理数）

### 实数的完整分类

$$
\text{实数} \begin{cases}
\text{有理数} \begin{cases} \text{整数（正整数、零、负整数）} \\ \text{分数（正分数、负分数）} \end{cases} \\
\text{无理数（正无理数、负无理数）}
\end{cases}
$$

> 💡 有理数 = 整数 + 分数；实数 = 有理数 + 无理数。

### 例题（中考必考）

**判断下列各数的类型：** $\sqrt{9},\ -\sqrt{5},\ \dfrac{22}{7},\ 0.3\dot{1},\ \pi,\ \sqrt[3]{27}$

- $\sqrt{9} = 3$ → **有理数**（整数）
- $-\sqrt{5}$ → **无理数**
- $\dfrac{22}{7}$ → **有理数**（分数！它只是 $\pi$ 的近似值，本身是有理数）
- $0.3\dot{1} = 0.3111\ldots$ → **有理数**（循环小数）
- $\pi$ → **无理数**
- $\sqrt[3]{27} = 3$ → **有理数**（整数）

**在数轴上定位 $\sqrt{2}$：**

因为 $1^2=1 < 2 < 4=2^2$，所以 $1 < \sqrt{2} < 2$。

又 $1.4^2=1.96 < 2 < 2.25=1.5^2$，所以 $1.4 < \sqrt{2} < 1.5$，约为 $1.414$。

## 🏐 扣球

### ❌ 易错区

| 陷阱 | 正确认识 |
|------|---------|
| 看到根号就是无理数 | $\sqrt{4}=2$ 是有理数！要看能否开尽 |
| $\frac{22}{7}$ 是 $\pi$ | $\frac{22}{7}$ 是有理数，$\pi$ 才是无理数 |
| 无限小数都是无理数 | $0.333\ldots=\frac{1}{3}$ 是有理数！无限**不循环**才是无理数 |
| $\sqrt{-4}$ 有意义 | 实数范围内，负数没有平方根 |

> 🎯 判断无理数三步：①能开尽吗？②含 $\pi$ 吗？③构造的无限不循环小数吗？满足任一→无理数
